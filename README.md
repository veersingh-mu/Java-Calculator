# Java Console Calculator
<br>

Simple command-line calculator built in Java for basic arithmetic operations. Perfect starter project demonstrating Scanner input, switch statements, and error handling.
<br>

## Features
<br>
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/) with division-by-zero protection
- Supports decimal numbers (double)

## How to Run
1. Save as `Calculator.java`
2. Compile: `javac Calculator.java`
3. Run: `java Calculator`
4. Enter numbers and operator as prompted

## Tech Stack
- Java (Scanner, switch, doubles)
- Console-based (no GUI)
